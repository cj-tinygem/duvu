# DUVU 템플릿 가이드

## 템플릿이란?

템플릿은 컬러 + 타이포 + 레이아웃 + 스타일 + 모션의 **종합 조합**이다.
사용자의 도메인/분위기에 맞는 최적의 프리셋 조합을 즉시 적용한다.

## 템플릿 메타데이터

각 템플릿은 다음 메타데이터를 가진다 (AI 추천 시스템용):
- **tags**: 검색/분류용 태그
- **industry**: 적합한 산업군
- **audience**: 타겟 사용자
- **complexity**: 구현 복잡도 (low/mid/high)
- **mood**: 분위기 키워드

상세 데이터는 `presets.json`의 `metadata.templates` 참조.
